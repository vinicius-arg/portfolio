HTML é uma linguagem de marcação que descreve o esqueleto de uma aplicação web. Com ela, busco aprimorar minhas competências acerca de boas práticas em acessibilidade e semantização de código.

Responsável por dar vidas às páginas estáticas, JavaScript foi minha primeira linguagem de programação real; devido à sua amplitude de uso, sempre há algo novo a aprender.

Banco de dados não-relacional, chamado NoSQL; utilizo seus serviços em conjunto com o Node.js, Express e Mongoose para a criação do back-end de sistemas web.

Node.js possibilita a execução de código JavaScript fora do navegador, é uma tecnologia que estou aprendendo em conjunto com Express para criar o back-end de aplicações.

Express é um framework para Node.js que facilita a criação de servidores. No momento, utilizo essa tecnologia para estudar desenvolvimento back-end.

Folhas de estilo em cascata, chamadas de CSS, dão estilo ao HTML. Atualmente objetivo desenvolver o uso de pré-processadores, como SASS e LESS.
